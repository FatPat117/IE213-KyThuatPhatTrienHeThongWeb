🪝 CUSTOM HOOKS CREATED

  1. useReadCampaignCount()
     → Total number of campaigns
     → Cache: 30 seconds

  2. useReadTotalRaised()
     → Total ETH raised (auto Wei→ETH conversion)
     → Returns both formatted and raw values

  3. useReadCampaign(id)
     → Specific campaign by ID
     → Null-safe, conditional fetching

  4. useReadAllCampaigns()
     → All campaigns array
     → Cache: 60 seconds (longer)

  5. useContractStats()
     → Combined count + raised
     → Single hook for dashboard

  6. useReadFilteredCampaigns(completed?)
     → Filtered campaigns (active/completed)
     → Client-side filtering

📈 CACHING STRATEGY

  Single Value Reads (campaignCount, totalRaised)
    ├─ Cache Duration: 30 seconds
    ├─ Refetch on: Window focus
    ├─ RPC Calls: ~2 per minute (normal usage)
    └─ Cache Hit Rate: ~95%

  Array Reads (allCampaigns)
    ├─ Cache Duration: 60 seconds
    ├─ Refetch on: Window focus (disabled)
    ├─ RPC Calls: 1 per minute
    └─ Cache Hit Rate: ~98%

  Manual Refetch
    ├─ Ignores cache
    ├─ Immediate update
    └─ Triggered by user action
